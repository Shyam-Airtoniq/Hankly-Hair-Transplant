# Hankly-Hair-Transplant

Demo landing page for **Euro Hair Transplant** — a Houston-based concierge connecting clients with premium European hair restoration clinics.

## Live Demo

Deployed on Vercel. The site is a single, self-contained `index.html` file with embedded CSS and JavaScript — no build step required.

## Stack

- Plain HTML / CSS / vanilla JavaScript
- Google Fonts: Montserrat, Open Sans, Cormorant Garamond
- Static images served from `/Images`
- Hero & mood imagery sourced from Unsplash (placeholder for demo — replace with licensed photography for production)

## Local Preview

Open `index.html` directly in a browser, or serve the directory:

```bash
npx serve .
```

## Deployment

The repository is configured for Vercel via `vercel.json`. Push to `main` and Vercel auto-deploys.

## Brand Guidelines

- **Royal Blue** `#0057B8` · **Navy** `#002147` · **Silver** `#C0C0C0` · **White** `#FFFFFF`
- Headlines: Montserrat (light weights for editorial feel)
- Body: Open Sans
- Serif accents: Cormorant Garamond italic
